Présentation Deezer

Description du Projet

Ce site présente l'entreprise Deezer, un service de streaming musical. L'interface utilisateur est moderne, mettant en avant les fonctionnalités clés de la plateforme.

Utilisation de gridbox, flexbox et animations CSS pour une expérience utilisateur fluide.

Le site est simple et coloré, afin de donné l'envie de s'y intéresser

Ce projet démontre ma maîtrise de la compétence "Réaliser un développement d'application" en utilisant les technologies web fondamentales (HTML5, CSS3, JavaScript). Développer une interface moderne avec des effets visuels, une architecture modulaire. Le site présente une approche professionnelle du développement front-end avec une attention particulière à l'expérience utilisateur et aux bonnes pratiques de développement web.

Technologies Utilisées

HTML5 CSS3 JavaScript
