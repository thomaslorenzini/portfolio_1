Site E-commerce

Description du Projet

Application de e-commerce de vente de Jean développé avec Flask lors d'une SAE.

La plateforme comprend le filtrage des produits, des options de tri, une fonctionnalité de panier.

Ce projet démontre ma maîtrise du développement web full-stack, l'implémentation d'une architecture backend avec des routes à l'aide de Flask en python. La gestion des sessions utilisateur avec l'aide des cookies, l'intégration d'une base de données relationnelle avec SQLite.

Ce projet illustre particulièrement la compétence "Gérer des données de l'information" à travers la conception et la gestion d'une base de données structurée pour les produits, utilisateurs et commandes, ainsi que l'implémentation de requêtes optimisées pour le filtrage et la recherche.

Technologies Utilisées

HTML5 CSS3 JavaScript Flask Bootstrap SQLite Python

Fonctionnalités Clés

  • Panier d'achat avec stockage persistant
  • Pages de détail des produits avec galeries d'images
  • Gestion des comptes utilisateurs
  • Suivi et historique des commandes
